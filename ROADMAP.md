## 🚧 Phase 1 — Contenu réel (court terme)

> Objectif : remplir le site avec nos vrais contenus

- [ ] **Galerie photos** — Remplacer les placeholders par de vraies photos (balades, bois, jardin, voyages)
- [ ] **Textes finaux** — Rédiger les textes définitifs pour chaque section
- [ ] **Favicon** — Créer un favicon bois/nature (feuille, tronc, initiales T&L)
- [ ] **Image hero** — Ajouter une photo de fond dans la section Hero
- [ ] **Open Graph** — Métadonnées pour le partage sur les réseaux sociaux (`og:image`, `og:description`)
- [ ] **Formulaire de contact** — Connecter à un service (Formspree, Netlify Forms, ou e-mail direct)

## 🌿 Phase 2 — Expérience enrichie (moyen terme)

> Objectif : rendre le site plus vivant et interactif

- [ ] **Blog / Carnet de bord** — Ajout d'une page d'articles (recettes, idées DIY, sorties nature)
- [ ] **Lightbox galerie** — Ouverture des photos en plein écran avec navigation
- [ ] **Menu hamburger mobile** — Navigation responsive sur mobile
- [ ] **Animation d'entrée** — Apparition douce des sections au scroll (`Intersection Observer`)
- [ ] **Curseur personnalisé** — Optionnel : curseur feuille ou gland 🍂
- [ ] **Mode sombre** — Variante nocturne aux tons bois sombres et bleu nuit

## 📅 Phase 3 — Fonctionnalités avancées (long terme)

> Objectif : transformer le site en vrai hub personnel

- [ ] **Calendrier d'événements** — Anniversaires, fêtes, projets à venir
- [ ] **Section "Wishlist" / Projets** — Idées maison, voyages rêvés, DIY à faire
- [ ] **Recettes** — Page dédiée avec filtres (végétarien, saison, durée)
- [ ] **Recommandations** — Livres, podcasts, films, artisans locaux
- [ ] **Newsletter** — S'abonner pour recevoir les actualités du site
- [ ] **Multilingue** — Version FR + EN pour les ami·e·s internationaux
- [ ] **PWA (Progressive Web App)** — Installer le site comme une appli sur mobile

---

## 🔒 Phase 4 — Technique & performance (en continu)

- [ ] **Optimisation images** — Format WebP, lazy loading, srcset responsive
- [ ] **Performance** — Score Lighthouse ≥ 90 (perf, accessibilité, SEO, best practices)
- [ ] **SEO** — Balises méta, sitemap XML, robots.txt
- [ ] **HTTPS** — Certificat SSL (via hébergeur ou Cloudflare)
- [ ] **Analytics respectueux** — Plausible ou Umami (sans cookies, RGPD-friendly)
- [ ] **Sauvegarde** — Versioning Git + déploiement automatique (GitHub Actions ou Netlify)
